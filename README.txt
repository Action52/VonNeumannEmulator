Gracias por usar el emulador de Von Neumann, la aplicación busca facilitar tu comprendimiento del lenguaje maquinal y ensamblador.

Para abrir la aplicación, ejecuta el archivo "Emulador.jar" ubicado en la carpeta "dist" de estos archivos.
Se incluye un ejemplo con la sintaxis propia de NUESTRO programa, es un programa en ensamblador que hace divisiones. El archivo se llama "ejemplo.txt" y se tiene que copiar y pegar su contenido en la casilla correspondiente del emulador para que funcione.
